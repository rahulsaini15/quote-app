# quote-app

This project is a React application that allows users to view and create quotes. Users can upload images associated with quotes and browse a list of quotes. It includes authentication, CRUD functionality for quotes, and an image upload feature
  ----

## Features

- User authentication with OTP login.
- View a list of quotes with images.
- Create and upload new quotes with images.
- Responsive design with mobile-first approach.

  ----

## Prerequisites

Before you begin, ensure you have met the following requirements:

- Node.js (version 16 or higher)
- npm (Node Package Manager)

If you don't have Node.js installed, download and install it from [nodejs.org](https://nodejs.org/).

You can verify your installation by running:

```bash
node -v
npm -v

  ----
